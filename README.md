# freelec-springboot2-webservice

## 개요

SpringBoot를 사용하여 웹서비스를 개발하고 AWS서비스를 활용하여 서비스를 배포하기 위해서 프로젝트를 진행하였다.  
CI/CD는 Github Actions로 구현하였다.
(현재 AWS 프리티어로 인하여 서비스를 모두 내려놓은 상태입니다.)

## 사용 기술

### SpringBoot
  - API
  - RESTAPI
  - JPA (MariaDB)

### Github
  - Action

### AWS
  - S3
  - EC2 (Amazon Linux2)
  - EIP
